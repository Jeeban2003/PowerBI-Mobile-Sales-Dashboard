# Mobile Sales Power BI Dashboard

This project presents a business intelligence dashboard built in Power BI to analyze mobile phone sales data.

## Tools Used
- Power BI
- Microsoft Excel
- SQL (for data understanding)

## Dashboard Features
- Total Sales, Quantity, Transactions, AOV and ASP
- Monthly Sales Trend
- Sales by Brand and City
- Customer Age Group Analysis
- Payment Method Distribution
- Brand Rating vs Sales

## Business Questions Answered
- Which mobile brand generates the highest sales?
- Which cities contribute the most revenue?
- Which age group buys the most?
- How do payment methods differ?
- How does customer rating impact sales?

## Purpose
This project was created to practice real-world business data analysis and dashboard development using Power BI.
